🚀 RAG ENGINE: Zero-Bloat Intelligence
The antidote to "Enterprise AI" theater.

Most enterprise AI projects are expensive, over-engineered PowerPoint decks. This repository provides a lean, high-performance RAG (Retrieval-Augmented Generation) pipeline that ditches the middleware bloat.

Built for speed. Optimized for grounding. Engineered for results.

🛠 The Stack
Engine: LangChain (Stable core abstractions)

Inference: Groq (Llama-3.3-70b-versatile for sub-second, elite-tier reasoning)

Vector Search: FAISS (Local, high-speed similarity search)

Embeddings: HuggingFace (all-MiniLM-L6-v2 for dense, efficient semantic representation)

Interface: Streamlit (Zero-latency UI)

⚡ Why This Matters
No Middleware Bloat: Direct ingestion to vector store. No unnecessary API hops or architectural overhead.

Contextual Integrity: Engineered chunking parameters (1000/200) designed to minimize hallucination by preserving document semantic structure.

Ready-to-Deploy: Minimalistic footprint allows for rapid prototyping or drop-in integration into existing workflows.
